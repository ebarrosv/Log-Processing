## Overview

Centralized logging and alerting stack:

- **Syslog sources** → rsyslog / syslog-ng on servers  
- **Fluentd** → aggregation, buffering, parsing, routing  
- **Graylog + ElasticSearch** → storage, search, dashboards  
- **Telegram** → real-time alerts via Graylog Notification Bot
